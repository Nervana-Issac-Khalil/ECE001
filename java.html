java

Java is a general-purpose, object-oriented, high-level programming language with several features that make it ideal for web-based development. WHAT MAKES LEARNING IT IMPORTANT? Java is used to develop enterprise-level applications for video games and mobile apps, as well as to create web-based applications with JSP (Java Server Pages). When used online, Java allows applets to be downloaded and used through a browser, which can then perform a function not normally available. Programs that use or are written in Java include Adobe Creative Suite, Eclipse, Lotus Notes, Minecraft and OpenOffice. Java is the core foundation for developing Android apps. Features: Application portability Robust and interpreted language Extensive network library

Importance of Java:

  • Platform Independent.
  • Java is to Facilitates Embedded Computing.
  • Java is Mature and Keeps Evolving.
  • Java is a Simplify Development of Real-Time Software.
